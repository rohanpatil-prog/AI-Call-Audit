
# What is this about
This system automatically audits insurance sales calls using AI. It converts call audio to text and analyzes conversations to detect miss-selling, compliance breaches, and script deviations by agents. The platform helps compliance and QA teams identify risk, improve agent quality, and ensure regulatory adherence at scale.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
